# CodeAlpha_Web-Scrapping-Books

## Project Overview

This project focuses on web scraping book information from an online bookstore website using Python. The scraper collects book titles, prices, ratings, and availability details from multiple pages and stores the data into a CSV dataset for analysis and visualization.

## Website Used

https://books.toscrape.com

## Objectives

* Perform web scraping on a real-world website
* Build a structured dataset from online data
* Practice automated data collection techniques
* Prepare data for analysis and visualization

## Tools & Technologies

* Python
* BeautifulSoup
* Requests
* Pandas
* Matplotlib
* Seaborn

## Features

* Scrapes multiple pages automatically
* Extracts:

  * Book Title
  * Price
  * Rating
  * Availability
* Saves data into CSV format
* Supports further EDA and visualization

## Dataset Information

The generated dataset includes:

* Title
* Price
* Availability Status
* Rating

## Project Workflow

1. Access website pages using Requests
2. Parse webpage HTML with BeautifulSoup
3. Locate book containers
4. Extract required information
5. Store collected data into a DataFrame
6. Export the final dataset as CSV

## Output

The project creates:

* A structured CSV dataset
* Visualization-ready data
